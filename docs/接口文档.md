# 接口文档

统一返回：

```json
{"code":200,"message":"success","data":{}}
```

## 登录

- `POST /api/auth/login`
- `GET /api/users/me`

## 活动

- `GET /api/activities`
- `POST /api/activities`
- `GET /api/activities/{id}`
- `PUT /api/activities/{id}`
- `DELETE /api/activities/{id}`

## 报名

- `POST /api/activities/{id}/register`
- `GET /api/activities/{id}/registrations`
- `GET /api/my/registrations`

## 签到

- `POST /api/activities/{id}/checkin`
- `GET /api/activities/{id}/checkins`
- `GET /api/activities/{id}/absences`
- `GET /api/my/checkins`

## 反馈

- `POST /api/activities/{id}/feedback`
- `GET /api/activities/{id}/feedbacks`

## 统计

- `GET /api/activities/{id}/stats`
- `GET /api/admin/dashboard`

## 工具

- `POST /api/activities/{id}/tools/draw`
- `POST /api/activities/{id}/tools/group`

## 导出

- `GET /api/activities/{id}/export/registrations`
- `GET /api/activities/{id}/export/checkins`
- `GET /api/activities/{id}/export/absences`
