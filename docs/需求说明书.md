# 活动魔方 MVP 需求说明书

## 目标

先完成可在本地演示的最小闭环：活动负责人创建活动，学生报名，学生签到，负责人查看名单和统计。

## 角色

- 学生 user：浏览活动、报名、签到、提交反馈。
- 活动负责人 organizer：创建活动、管理报名、生成签到码、查看统计、抽签分组、导出名单。
- 管理员 admin：具备活动负责人能力。

## MVP 范围

- 登录：账号密码登录，返回简化 token。
- 活动：列表、详情、创建、编辑、删除。
- 报名：重复报名校验、报名时间校验、人数上限校验、跨校区限制。
- 签到：已报名才可签到，重复签到拦截。
- 统计：报名人数、签到人数、未签到人数、签到率、按校区统计。
- 工具：随机抽签、随机分组。
- 导出：报名名单、签到名单、未签到名单 CSV。

## 非目标

- 不接入学校统一认证。
- 不做真正微服务拆分。
- 不在代码中写真实数据库密码。
